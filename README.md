## Run Locally

Clone the project

```bash
  git clone https://github.com/Kisari/SEPM---web-development
```

Go to the client project directory

```bash
  cd client
```

Install dependencies

```bash
  npm install --force
```
Go to the server project directory

```bash
  cd server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
