## Run Locally

Clone the project

```bash
  git clone https://github.com/Kisari/SEPM---web-development
```

Go to the server of project directory

```bash
  cd server
```

Install dependencies

```bash
  npm install
```
Start server

```bash
  npm run start
```

Install client dependencies

```bash
  cd client
```

Install dependencies

```bash
  npm install --force
```

Start client 

```bash
  npm start
```
